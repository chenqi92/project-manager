项目环境管家（Project Env Manager）· 隐私政策

生效日期：2026-06-19

「项目环境管家」是一个本地优先、端到端加密的 Chrome 凭据管理扩展，用于管理多项目、多环境、多平台的登录凭据。

1. 我们收集哪些数据

  • 你创建的凭据：用户名、密码、链接地址、备注、TOTP 密钥。
  • 应用设置 / 偏好（自动锁定时长、同步配置、主题等）。
  • 我们不收集浏览历史、分析、遥测、IP 日志，没有任何数据发送到开发者的服务器——开发者不运营任何服务器，不接收任何用户数据。

2. 数据如何被使用

仅用于在你自己的设备上存储、填充、管理你的凭据，无其它用途。

3. 存储与加密

全部金库数据在写入磁盘前先在本地加密：AES-256-GCM，密钥由你的主密码经 Argon2id / PBKDF2 派生。主密码与派生密钥永不离开你的设备。明文凭据仅在金库解锁期间短暂存在于内存。

4. 可选的自托管同步

同步默认关闭，需手动开启。开启后，扩展只会把已加密的金库（密文）上传到你自己配置并运营的同步服务器，全程 HTTPS/TLS。服务器只存密文，运营者无法解密你的数据。你可随时关闭同步并删除远端副本。

5. 数据共享与披露

我们不出售、不共享、不向任何第三方转移你的数据。无广告、无数据经纪、无分析合作方。共享：无。

6. 数据保留

数据在本地持续保存，直到你删除或卸载扩展。开启同步时，密文在你的服务器上保留直到你删除。卸载将清除本地数据。

7. 你的权利与控制

你可在扩展内随时查看、编辑、导出、删除全部数据，无需向我们提出请求。

8. 政策变更与联系方式

任何实质性变更都会更新本页的生效日期。如有疑问或数据相关请求，请联系：请替换为你的联系邮箱。

Limited Use 声明

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements. All credential data is end-to-end encrypted on the user's device, is used solely to provide the extension's password-management function, is never sold or transferred to third parties, is never used for advertising, and is never read by the developer (the developer operates no servers; where optional sync is enabled, the operator stores only ciphertext).
