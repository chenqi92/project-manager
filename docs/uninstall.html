项目环境管家 · 卸载与数据恢复

你已卸载本扩展，或正在查看卸载相关说明。

重要：Chrome 在卸载扩展时会立即清空该扩展的本地数据。 这是浏览器的统一行为，并非本扩展可以阻止。如果你没有事先导出备份、也没有开启同步， 被卸载时清除的数据将无法找回——由于全部数据端到端加密、不上传任何第三方服务器， 开发者同样无法帮你恢复。

升级不需要卸载重装

Chrome 扩展会自动从应用商店更新，更新过程保留你的全部数据。 你无需、也不应该为了升级而卸载后重新安装——那样会丢掉数据。

如果你之前开启了同步

数据可以恢复：

  1. 重新安装本扩展。
  2. 在解锁页选择「已有同步保险箱？从同步服务器恢复」。
  3. 填入你的同步服务器地址与令牌，拉取后用原主密码解锁即可。

如果你之前导出过加密备份

数据可以恢复：

  1. 重新安装本扩展，创建（或解锁）保险箱。
  2. 进入「导入 / 导出 → 导入」，来源格式选「加密备份」。
  3. 选择此前导出的备份文件，输入备份密码导入。

以后如何避免再次丢失

  • 开启自托管同步：密文实时保存在你自己的服务器，换机/重装都能恢复。
  • 定期导出加密备份：备份文件即使泄露也无法被解密，请妥善保管。
  • 换电脑或重装系统前，务必先完成上述任一项。
